# 安装场景档案

只记录会影响安装路线的字段。运行 `chm-agent scenarios <模型目录>`，获取所选产品版本的准确字段 ID 和允许值。文档或用户未提供取值时，使用 `待确认`。

## 主路线选择条件

| 字段 | 示例值 | 作用 |
|---|---|---|
| 产品版本 | 7.3.0 | 选择版本化模型及其证据集 |
| `installation_nature` | `new`、`add_on` | 全新安装和追加部署路线不得混用 |
| `deployment_carrier` | `physical`、`virtual` | 选择主要安装章节 |
| `base_environment` | `none`、`smart_datacube`、`campaign`、`smart_decision`、`wdtn` | 选择追加部署路线 |
| 组网类型 | 标准、轻量化、工具、已有产品追加部署 | 影响节点数、可靠性和限制 |
| 已有产品 | Decision、DataCube、Campaign、WDTN、GDE | 选择追加部署和集成过程 |
| 硬件 | 厂商硬件、第三方硬件 | 影响环境评估和操作系统准备 |
| 操作系统及版本 | EulerOS、受支持的第三方操作系统 | 决定支持范围和整改步骤 |
| CPU 架构 | x86_64、ARM64 | 影响软件包、客户端和受支持的转换路径 |
| 数据平台 | 内置 FI、外置 FI、其他 Hadoop、无 | 影响资源准备和网络配置 |
| 数据平台访问方式 | admin、OM、非 OM | 选择用户和客户端准备路径 |
| 数据库 | 内置、MySQL、庞威、其他外置数据库 | 选择数据库准备和参数配置过程 |
| 数据库 SSL | 启用、禁用 | 选择互斥的数据库步骤 |
| 网络 | 平面规划、IPv4/IPv6、平面合设 | 影响 IP、VIP、路由和限制 |
| 可选能力 | Campaign、DataCube 设计态、模型、资产 | 增加条件分支 |

## 次要规划输入

- 节点数量和规模；
- 在线、离线或受代理限制的交付环境；
- 跳板机类型和可用执行账号；
- 存储、分区、备份服务器、DNS 和 NTP；
- 证书来源、License、软件包、APP 和场景包的可用性；
- 停机窗口、备份状态和回滚要求。

## 完整性判定

对每个缺失字段进行分类：

- `阻断`：不同取值会选择不同的主流程。生成可执行的安装执行手册前必须询问用户。
- `条件化`：分别给出带标签的分支，然后继续规划。
- `非阻断`：记录到待确认问题中，不延迟方案输出。

对于“生成物理机场景安装流程”这类请求，设置 `deployment_carrier=physical`，编译不完整的场景档案，并使用编译器返回的阻断项和条件步骤，不得自行假设其他取值。
